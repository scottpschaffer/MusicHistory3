# Music History 3

### Description:
> The purpose of this project is to modify the Music History program to create a List Mode (where the songs are all listed) and a Add Mode (where additional songs can be added to the list).

### How to run:
```
http-server -p 8888
```

> This will show at:
`http://localhost:8888`
